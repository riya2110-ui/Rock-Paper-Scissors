# Rock-Paper-Scissors

A simple browser-based Rock-Paper-Scissors game built with HTML, CSS, and JavaScript. Users click a move to play against a randomly chosen computer move. Scores update live and the result message shows the match outcome.

## Features

- Clickable **rock**, **paper**, or **scissors** choices.
- Computer selects a random move.
- Live tracking of user and computer scores.
- Result message updates for wins, losses, and draws.
- Responsive, centered UI with hover effects.

## Files

- index.html — game user interface and layout.
- style.css — styling for the game board and controls.
- app.js — game logic, score tracking, and event handling.
- images/ — icons for rock, paper, and scissors.

## How to Use

1. Open index.html in a browser.
2. Click one of the three choices: rock, paper, or scissors.
3. View the result message and score updates.
4. Play again to continue the match.
